# DeepLearning-Assignment3

This Assignment is implemented building a simple fully connected Deep Network from scratch with following:

- BackPropagation & GradientPropagation
- Non-Linear Regression without using pytorch
- Non-Linear Regression with using pytorch
- PyTorch Lightening
- TensorFlow variants
